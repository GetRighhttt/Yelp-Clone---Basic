# Yelp-RandomUser-GoogleMaps
Using Yelp api to return a list of restaurants, and a random users API to generate users. Also added in Google maps for the restaurant location.

#### Here is a list of all the features used in this application:

- Google Maps
- Swipe Mechanism
- Design Patterns
- Retrofit2
- API(2)
- Coroutines
- Dagger-Hilt
- MVVM
- StateFlows
- Coroutines
- Sealed Classes
- Intents: PutExtra
- Glide
- Recycler View
- Animations
- View Binding
- Material3
- Material Dialogs
- Material Toolbar
- Snackbar
- Splash Screen
